  • defination of web designing languages
    • HTML
    HTML is a HyperText Markup Language. It is a language that displays information in hypertext format.
    Using HTML, we can design web pages & construct websites.
    XML
    XML is an eXtensible Markup Language. It is used in different ways and is quite different from HTML.
    XML is an improvement to SGML such that its main objective is structuring documents. In XML, we use user-defined tags.
    JAVASCRIPT
    JAVASCRIPT is used to produce dynamic web pages. JavaScript uses HTML tags & programming facilities. JavaScript uses HTML tags for web designing & programming facilities to produce dynamic webpage contents.
